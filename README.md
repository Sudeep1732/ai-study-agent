📚 AI Study Assistant Agent

🚀 Turning AI into your personal tutor

---

🧠 About

AI-powered study assistant that helps students understand concepts, summarize topics, and learn faster using natural language.

---

🚀 Features

- 🤖 Ask study questions
- 📄 Summarize topics
- 🧠 Simple explanations
- 💬 Chat-based interaction
- 🌐 Web interface using Streamlit

---

🛠️ Tech Stack

- Python
- Streamlit
- OpenRouter / OpenAI API

---

▶️ How to Run

pip install openai streamlit
py -m streamlit run app.py

---

🔑 Setup

Add your API key in app.py:

api_key="YOUR_API_KEY"

---

🎯 Use Case

- Students learning concepts
- Exam preparation
- Quick revision
